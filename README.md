# MonadBoard - Decentralized Billboard

A decentralized billboard application built on the Monad Testnet where users can bid to display their message. If outbid, the previous owner receives an instant 5% profit.

## Features

- **Buy Spot**: Connect wallet and bid to own the billboard.
- **Viral Economics**: Previous owners get paid when outbid.
- **Live Feed**: Real-time updates of network activity (Simulated).
- **Leaderboard**: Top performers tracking (Simulated).
- **Smooth UX**: Premium animations, glass-morphism UI, and intuitive modals.

## Tech Stack

- **Frontend**: HTML5, Vanilla JavaScript, TailwindCSS (CDN)
- **Blockchain Interaction**: Ethers.js
- **Network**: Monad Testnet

## Setup

1. Clone the repository.
2. Open `index.html` in your browser (or use Live Server).
3. Connect your Monad Testnet wallet.
4. Start bidding!
